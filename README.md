## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Java GUI system for storing and manipulating customer information, with functionality to add items and make purchases. 

### Built With

* Java
* Netbeans IDE

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* Install [Java](https://java.com/en/download/help/download_options.html)

### Installation

Download and run the `.jar` latest version of the application in the [Releases](https://github.com/Tim-W-James/Java-Client-Management-System/releases) tab.
   
### Development

Use the Netbeans IDE.
